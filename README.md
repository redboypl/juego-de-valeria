# Juego de Razones Trigonométricas

Un juego dinámico basado en navegador para practicar las razones trigonométricas (seno, coseno, tangente) con ángulos especiales de 30°, 45° y 60°. Incluye animaciones, un temporizador de 10 segundos por pregunta, seguimiento de racha y finales bueno/malo con 10 preguntas para completar.

## Características
- Diseño amigable para móviles con animaciones fluidas
- Preguntas aleatorias sobre seno, coseno, tangente con ángulos de 30°, 45° y 60°
- Formato de opción múltiple con 4 opciones (1 correcta, 3 incorrectas)
- Temporizador visual con barra de progreso
- Seguimiento de puntuación y racha de respuestas correctas consecutivas
- Progresión con 10 preguntas para completar el juego
- Finales bueno (completar todas) y malo (fallar una pregunta)
- Retroalimentación visual con colores y animaciones
- Soporte para entrada táctil y de teclado

## Cómo Jugar
1. Abre `index.html` en tu navegador web.
2. Responde 10 preguntas trigonométricas seleccionando una de las 4 opciones antes de que se acabe el tiempo (10 segundos).
3. Haz clic en la opción que creas correcta.
4. Si respondes correctamente, avanzas; si fallas o se acaba el tiempo, pierdes y puedes reiniciar.
5. Completa todas las preguntas para ganar el juego.

## Requisitos
- Un navegador web moderno con soporte para CSS animations (Chrome, Firefox, Safari, Edge)

## Desarrollo
- `index.html`: Estructura HTML con elementos dinámicos
- `style.css`: Estilos avanzados con animaciones y gradientes
- `script.js`: Lógica del juego con temporizador y efectos visuales

Para ejecutar localmente, abre `index.html` en un navegador.
